üì° API Endpoints

GET Get All Opportunities

/api/opportunities

Returns all job opportunities with optional filtering

Query Parameters: category, subcategory, type, experience

GET Get Opportunities by Category

/api/opportunities/category/{category}

Returns opportunities filtered by category (tech, career, education, managers, non-tech)

GET Get Single Opportunity

/api/opportunities/{id}

Returns a specific opportunity by ID

GET Get All Categories

/api/categories

Returns all available job categories with descriptions

GET Search Opportunities

/api/search?q={term}

Search opportunities by title, company, description, or skills

GET Get Statistics

/api/stats

Returns statistics about opportunities and categories

GET Get Companies

/api/companies

Returns list of all companies with opportunities

GET Get Opportunities by Company

/api/companies/{company}/opportunities

Returns all opportunities for a specific company
